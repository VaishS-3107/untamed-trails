# untamed-trails
Untamed Trails is a multi-page travel website built using HTML, CSS, and JavaScript to inspire and guide users through solo, family, and group travel experiences.

# Technologies Used

- HTML5
- CSS3
- JavaScript

# Pages Included

- Home
- About
- Travel Styles
- Solo Travel
- Family Trips
- Friends & Groups
- Blog
- Gallery
- Reviews
- Contact

# Purpose

This project was developed as part of a web development academic project to demonstrate front-end development skills including website structure, navigation, layout design, and user interaction.
